# 📖 Overview

AI Recipe Generator is a smart web application that helps users discover recipes from the ingredients they already have at home. The application provides AI-powered recipe suggestions through an intuitive and responsive interface.

The project also includes voice input, recipe analytics, and local storage functionality, making meal planning faster, easier, and more interactive.

---

# ✨ Features

* 🍅 Generate recipes from available ingredients
* 🤖 AI-powered recipe recommendations
* 🎤 Voice input using Web Speech API
* 📊 Dashboard with recipe statistics (Chart.js)
* 💾 Save recipe history using Local Storage
* 🔍 Search recipes quickly
* 📱 Responsive design for desktop and mobile
* ⚡ Fast and user-friendly interface

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/AI-Recipe-Generator.git
```

Move into the project folder.

```bash
cd AI-Recipe-Generator
```

Create a virtual environment.

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux/Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

Install dependencies.

```bash
pip install -r requirements.txt
```

Run the Flask application.

```bash
python app.py
```

Open your browser.

```
http://127.0.0.1:5000
```
